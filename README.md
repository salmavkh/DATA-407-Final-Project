# DATA 407 Final Project

This repository contains the R Markdown source file and rendered HTML report for my DATA 407 project on AI startup reviews.

## Files
- `DATA407_FinalProject.Rmd` — main R Markdown report
- `DATA407_FinalProject.html` — rendered HTML output

## Project Focus
The report examines three questions:
1. Sampling bias in review positivity after adjusting for startup-cluster imbalance
2. Topic distribution and satisfaction across startup clusters
3. Emotion intensity differences between extreme and non-extreme ratings

The analyses were conducted in **RStudio using R**.

## Related Project
This report uses data and preprocessing pipelines developed as part of my broader honours thesis project on AI startup reviews, which can be found here: https://github.com/salmavkh/ai-startup-reviews.
